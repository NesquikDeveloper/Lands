# Minecraft Network System
Sistema de plugins e ferramentas para redes de **Minecraft**, disponibilizado publicamente neste repositório para estudo, uso autorizado e contribuições da comunidade.

## ⚠️ Autorização obrigatória para uso
Para que uma rede de **Minecraft** possa utilizar este sistema, o **endereço IP (IPv4) ou porta do servidor precisa estar previamente autorizado pelo proprietário deste repositório** ou pela equipe de moderadores.

O sistema pode utilizar verificações internas baseadas em:
* IPv4 externo da máquina
* Porta primária do servidor
* Configurações definidas no `config.json`

Caso o servidor **não esteja autorizado**, os plugins e sistemas podem:
* Não iniciar corretamente
* Bloquear funcionalidades internas
* Impedir que a rede seja iniciada

Mesmo que alguém tente **utilizar atualizações ou versões modificadas sem autorização**, o sistema pode impedir o funcionamento da rede.

---

## ❗ Responsabilidade
Caso uma rede fique **offline ou bloqueada por falta de autorização**, **isso não é responsabilidade do desenvolvedor ou do repositório**.

Se alguém denunciar comportamentos inadequados ou má utilização da rede que esteja usando este sistema **sem autorização**, a responsabilidade é **inteiramente do administrador dessa rede**.

Cada proprietário de rede deve:
* Seguir as regras básicas abaixo
* Manter conduta adequada em sua comunidade
* Administrar seu próprio servidor de forma responsável

---

# 📜 Regras de uso
## Regra 1 — Proibição de venda
É **proibido vender qualquer plugin ou sistema presente neste repositório**.

Qualquer venda sem autorização será considerada **venda clandestina**.

---

## Regra 2 — Licenciamento
Não solicite ao proprietário para **privar o GitHub ou vender licenças especiais**.
Este sistema **não está à venda**.
Dinheiro **não compra autorização para violar estas regras**.

---

## Regra 3 — Conduta da rede
Não será permitido manter redes com comportamentos considerados prejudiciais ou ilegais.

Entre eles:
* Racismo
* Xenofobia
* Apologia ao uso de drogas
* Apologia ao nazismo
* Apologia a movimentos terroristas

Consequências possíveis:
* Banimento da rede
* Bloqueio do IP
* Bloqueio do endereço
* Bloqueio do nome da rede

Em casos graves, os **nomes dos proprietários podem ser adicionados a uma blacklist pública** para evitar que enganem outras comunidades.

---

## Regra 4 — Conduta administrativa
Não será tolerada **imaturidade administrativa**.

Isso inclui:
* Pedir para amigos da staff removerem servidores sem motivo
* Solicitar punições sem provas
* Uso abusivo do sistema administrativo

Qualquer denúncia deve conter **provas claras**, como:
* Capturas de tela completas
* Gravações de tela completas
* Sem cortes ou telas escuras

---

## Regra 5 — Desenvolvimento personalizado
Não solicite ao proprietário para desenvolver **sistemas exclusivos para sua rede gratuitamente**.

Se deseja algo exclusivo para sua rede, **pagamento deve ser acordado previamente**.
> A vida é triste, mano.

---

## Regra 6 — Plugins privados
Alguns plugins podem ser:
* Públicos
* Privados
* Liberados mediante autorização

Plugins públicos podem receber:
* Atualizações semanais
* Atualizações mensais

Plugins privados podem exigir **pagamento ou autorização vinculada à `config.json` do servidor**, dependendo do caso.

---

# 📦 Configuração
A autorização do servidor pode ser baseada nas informações definidas em:
```
config.json
```

Exemplo de campos utilizados:
* `ipv4`
* `port-primary`
* `port-important`

Essas informações podem ser usadas para validar o acesso da rede ao sistema.
